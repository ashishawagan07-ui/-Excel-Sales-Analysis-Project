\# Sales Dashboard in Excel



\## Project Overview

This Sales Dashboard was created in Microsoft Excel to analyze and visualize sales data using charts, KPIs, and interactive filters. The dashboard helps track sales performance, revenue trends, and business insights.



\## Features

\- Sales Performance Analysis

\- Revenue and Profit Tracking

\- Interactive Filters (Slicers)

\- KPI Monitoring

\- Data Visualization using Charts



\## Tools Used

\- Microsoft Excel

\- Pivot Tables

\- Pivot Charts

\- Slicers

\- Data Cleaning
